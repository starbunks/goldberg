== Description ==

This plugin allows you to embed Facebook posts everywhere in your Wordpress site. Simply use the Facebook button on the page/post editor.
Alternatively, you can embed Facebook posts by adding the URL for the post to the plugin shortcode:
`[fb_embed_post href="http://www.facebook.com/link-to-post" width="550"/]`

== Installation ==

**Install Normally**

Simply search for 'Facebook Posts Embedded' in the Plugins Admin page, then install and activate it. That's it!

**Install as Plugin**

1. Upload `wp-facebook-posts-embedded` folder into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activating the plugin, you will find the Facebook button in any page/post editor.

== Frequently Asked Questions ==

= How can I embed Facebook posts into my page/post? =

Simply use the Facebook button on the page/post editor.
Alternatively, you can embed Facebook posts by adding the URL for the post to the plugin shortcode:
`[fb_embed_post href="facebook-link"/]`

= Why should I download this plugin? =

You should download it if you need to embed Facebook posts into your WordPress site through a button in the editor.

= Can the Facebook post be customized? =

Unfortunately Facebook does not allow this practice yet.
